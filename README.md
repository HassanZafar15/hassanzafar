# Hassan Zafar — Portfolio

Personal portfolio website for **Hassan Zafar**, Senior Flutter Developer and Mobile Architect.

**Live site:** [hassanzafar.netlify.app](https://hassanzafar.netlify.app/)

## About

A modern, responsive portfolio built with React and Vite. It showcases professional experience, technical skills, featured projects, Upwork client testimonials, and contact information.

### Sections

- **Hero** — Introduction, CV download, and social links
- **About** — Professional journey and key metrics
- **Skills** — Flutter, backend, payments, and DevOps proficiency
- **Experience** — Career timeline
- **Projects** — Featured apps and platforms (iOS, Android, Flutter Web)
- **Testimonials** — Client reviews from Upwork
- **Contact** — Email, phone, and social profiles

## Tech Stack

| Layer | Technology |
|-------|------------|
| Framework | React 19 + TypeScript |
| Build tool | Vite |
| Styling | Tailwind CSS v4 |
| Animations | Motion (Framer Motion) |
| Icons | Lucide React |

## Getting Started

**Prerequisites:** Node.js 18+

```bash
# Clone the repository
git clone https://github.com/HassanZafar15/hassanzafar.git
cd hassanzafar

# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start local dev server on port 3000 |
| `npm run build` | Build for production (`dist/`) |
| `npm run preview` | Preview production build locally |
| `npm run lint` | Run TypeScript type check |

## Deployment

This is a static site. Build and deploy the `dist/` folder to any static host (e.g. Netlify, Vercel, GitHub Pages).

```bash
npm run build
```

## Security

This is a **public repository**. Do not commit secrets or credentials.

- Environment files (`.env`, `.env.local`) are excluded via `.gitignore`
- Use `.env.example` as a reference for optional local configuration only
- Never push API keys, tokens, or private URLs to this repo
